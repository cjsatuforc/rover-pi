# Raspberry RoverPi

This is the code source for a Rover Pi Drone project.

## Description

We used a Raspberry Pi connected to motorized wheels using [H bridges](https://en.wikipedia.org/wiki/H_bridge).

You will find here the code used to manage GPIOs and close bridges to move the rover.
